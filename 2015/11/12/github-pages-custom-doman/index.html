Github Pages绑定域名

2015-11-12 | Branger<br/>生活因你而精彩 |

Github Pages的默认域名是github的二级域名username.github.com或者username.github.io，需要绑定自己的独立域名的话也非常简单，只要以下两步：

一、增加CNAME

在你的git仓库创建一个CNAME文件，内容写你的域名。比如我的域名是qy1195.com，在CNAME文件写入“qy1195.com”就好了。
